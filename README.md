# Space Buds 

Space Buds team repository for NASA's 2021 International Space Apps Challenge - "Let It Go (Without A Bang)".

Please find our pitch video [here](https://www.youtube.com/watch?v=mZa6cD9xq2c), or feel free to browse this repository for more information. This README provides an overview of the information contained within the repository.
