# Space Buds 

Space Buds team repository for NASA's 2021 International Space Apps Challenge - "Let It Go (Without A Bang)".

Please find our pitch video [here](https://www.youtube.com/watch?v=mZa6cD9xq2c), or feel free to browse this repository for more information. This README provides an overview of the information contained within the repository.

We would also like to highlight our design report, which can be found under `Documentation/report.pdf`.

## Documentation Files
File | Contents
---- | --------
`Documentation/report.pdf` | A complete design report detailing every aspect of our project. Includes renders, drawings and references.
`Documentation/presentation.pdf` | Our 5-slide pitch presentation, presented [here](https://www.youtube.com/watch?v=mZa6cD9xq2c).
`Documentation/Datasheets/Electrical/` | Datasheets for all off-the-shelf mechanical components used in our design.
`Documentation/Datasheets/Mechanical/` | Datasheets for all off-the-shelf electrical components used in our design.

## Mechanical Files
File | Contents
---- | --------
`Mechanical/animation.mp4` | An animation of the system deploying.
`Mechanical/Drawings/SolidWorks/` | SolidWorks mechanical drawings files.
`Mechanical/Drawings/Images/` | SolidWorks mechanical drawings as JPEG or PDF files.
`Mechanical/Models/SolidWorks/` | SolidWorks 3D model files.
`Mechanical/Renders/` | Mechanical part renders.

## Electrical Files

File | Contents
---- | --------
`Electrical/KiCad/` | KiCAD electrical CAD files for the PCB.
`Electrical/Renders/` | Renders of the PCB, generated by KiCAD.
`Electrical/layout.png` | An image of the PCB's layout.
`Electrical/schematic.png` | An image of the PCB's schematic.
